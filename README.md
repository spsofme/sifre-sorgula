# Şifre Sorgula

Kullandığınız şifreleri hazır şifre dosyalarından çekilen şifreler arasında sorgulamak için aracı bir program.

## Kurulum

1. `git clone https://github.com/spsofme/sifre-sorgula.git`
2. `cd sifre-sorgula`
3. `pip install -r requirements.txt`

## Programın kullanımı

`python3 ./sorgula.py`
